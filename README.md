# Smart Attendance System

## Introduction

This project is a Smart Attendance System developed to make attendance easier and faster. Instead of taking attendance manually, this system uses WiFi and a captive portal to record student attendance automatically. Students can connect using their own mobile phones or laptops.

## Purpose

The main purpose of this project is to reduce manual attendance work and save time. It also helps teachers to manage attendance records easily.

## How the system works

1. The system creates a WiFi network.
2. Students connect to the WiFi using their devices.
3. A login page opens automatically.
4. Students enter their details.
5. The system records their attendance.
6. Attendance can be saved and downloaded later.

## Features

* Automatic attendance recording
* No need for manual attendance
* Easy to use
* Saves time
* Attendance data can be stored
* Supports mobile phones and laptops

## Technologies used

* HTML
* CSS
* JavaScript
* WiFi and Captive Portal
* CSV file for storing attendance

## Advantages

* Fast and simple attendance process
* Reduces human errors
* No physical contact required
* Easy to maintain records

## Conclusion

This Smart Attendance System is useful for schools and colleges. It makes attendance faster, easier, and more efficient compared to the traditional method.
